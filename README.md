# minishop
Spring Boot Frameworkを活用して練習するミニプロジェクト。

## プロジェクト目的
本、プロジェクトは、Spring Bootを使って簡単なミニショップを作ってみるプロジェクトです。 
主な機能は、会員、商品、注文、カートに構成された小さいプロジェクトになります。

## 使用言語
- Java 11
- html 5 
- css 3 
- JavaScript ES6

## フレームワークおよびライブラリ
- SpringBoot 2.7.15
- Thymeleaf
- gradle
- lombok
- Swwetalert2
- MyBatis
